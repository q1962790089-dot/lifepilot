# Chat Style V1.2 手动测试

每条输入请使用新的空白聊天会话，避免上一条内容影响结果。

| # | 输入 | 预期 |
| --- | --- | --- |
| 1 | 我又点外卖了，68块 | 保存 expense；回复 1-2 句，不擅自安慰、评判或分析消费动机。 |
| 2 | 今天跑了三公里 | 保存 exercise；自然短确认，不延伸习惯分析。 |
| 3 | 明天十点开会 | 保存 todo 和 dueDate；自然短确认。 |
| 4 | 今天体重70公斤 | 保存 weight；不评价身材或推测情绪。 |
| 5 | 今天好累 | 自然接话，可轻微关心或调侃，不解释疲惫原因。 |
| 6 | 我今天什么都没干 | 短回复，不使用“批准”“不批评”等上位者措辞。 |
| 7 | 不想上班 | 自然接话，不写鸡汤或强制反问。 |
| 8 | 今天又被领导骂了 | 可有限度站队，不把对方动机当事实。 |
| 9 | 别分析，我就是想骂他 | 直接接住，不复述指令或总结心理状态。 |
| 10 | 你为什么这么冷淡 | 简短承认并改语气，不写复盘报告。 |
| 11 | 你说话好像机器人 | 简短承认并改语气，不解释内部策略。 |
| 12 | 我明天到底该怎么和领导说 | 先给可执行方案，不先索要完整背景。 |
| 13 | 我该怎么回复他 | 先给通用回复思路，最多补一个有意义的问题。 |
| 14 | 家里人生病了，我很难受 | 不玩梗、不调侃；安静、清楚、有支持感。 |

通用检查：

- 不展示内部意图分类、情绪强度或自检过程。
- 不虚构用户没有表达的情绪，不重复最近上下文，不为续聊提出无意义问题。
- 不出现“批准你”“不批评你”“原谅你”“罪名不成立”“现在先不分析”等表达。
- 清醒型、温柔型、亲密型只修饰语气，不覆盖意图规则。
- 所有记录类型继续写入 `lifepilot_records`，且 `/api/chat` 响应结构不变。
